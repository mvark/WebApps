# WebApps
Web Apps that I've created, enhanced or remixed through my collaboration with GitHub Copilot -
1. [AltTitleTagGenerator](https://github.com/mvark/WebApps/tree/main/AltTitleTagGenerator) — Bookmarklet for generating ALT & TITLE tags through Gemini AI after letting it process an image.
2. [AskUTube](https://github.com/mvark/WebApps/tree/main/AskUTube) — Simple web tool to extract transcript of a specified YouTube video and answer questions based on that content.
3. [BingeBuddy](https://github.com/mvark/WebApps/tree/main/BingeBuddy) is a light weight web application for searching movies, TV shows, and celebrities with streaming availability in India. Co-created with GitHub Copilot using vanilla JavaScript and powered by The Movie Database (TMDB) API.
4. [Bookmarker](https://github.com/mvark/WebApps/tree/main/Bookmarker) is a combination of Bookmarklet code & a Google App Script built API that saves bookmarks while automatically categorizing URLs using Gemini Flash API.
5. [Price & Expiry Date AI Extractor](https://github.com/mvark/WebApps/tree/main/DateExtractor) - A lightweight, single-file web application that uses Google Gemini AI to extract and validate retail data from photos of packaged food.
6. [ELI5 Text Explainer](https://github.com/mvark/WebApps/tree/main/ELI5) - A minimal, single-page web app that takes any text and explains it “like I’m 5”, using free large-language models available via OpenRouter.
7. [Export As Word Document](https://github.com/mvark/WebApps/tree/main/ExportAsWord) code sample demonstrates how to generate and download a Word document (.doc) file directly from content on a webpage using only JavaScript and HTML. The entire file creation process happens on the client side.
8. [Export As Word Document Bookmarklet](https://github.com/mvark/WebApps/tree/main/ExportAsWordDocBookmarklet) Bookmarklet variant for exporting content as Word documents
9. [HTML to PDF](https://github.com/mvark/WebApps/tree/main/HTML2PDF)  - A simple, lightweight, 100% client-side web tool that lets users highlight any text on the page and export only the highlighted portion as a clean PDF
10. [MF-Charts](https://github.com/mvark/WebApps/blob/main/MF-Charts) — Charting tool that visualizes the multi‑year NAV history of an Indian mutual fund using the free MFapi.in API. It is designed to help investors study seasonal patterns and see 52‑week high/low levels for a scheme at a glance.
11. [NPS-Charts](https://github.com/mvark/WebApps/blob/main/NPS-Charts) — Charting tool that visualizes multi‑year NAV history of schemes under National Pension System (NPS)
12. [NutriExtract](https://github.com/mvark/WebApps/blob/main/NutriExtract) — Web app to extracts nutritional information from image of nutrition table using Gemini AI API.
13. [NutriScan](https://github.com/mvark/WebApps/tree/main/NutriScan) uses the Open Food Facts API to detect whether a scanned food item is Ultra Processed or not.
14. [People Tracker](https://github.com/mvark/WebApps/tree/main/PeopleTracker) is a simple bookmarklet that allows you to quickly save the URL, title, and your personal notes about any webpage (like a social media profile) directly into a Google Sheet. Your data stays in your own sheet and you can keep editing it whenever you like
15. [PhonaTick](https://github.com/mvark/WebApps/tree/main/PhonaTick) is a front-end for a list of words with tricky pronunciations fetched via a JSON API through Datasette
16. [Protein Price Calculator](https://github.com/mvark/WebApps/tree/main/ProteinPriceCalculator) — Calculator for comparing protein content by price.
17. [GitHub User Roaster](https://github.com/mvark/WebApps/blob/main/Roaster/README.md)  is a Streamlit app that fetches a GitHub user’s metadata via the GitHub API, analyzes recent public commits—highlighting coding patterns, quirky commit messages, and even emojis—and then uses Perplexity AI’s Sonar model to deliver a sharp, humorous “roast” of the user.
18. [Things Near Me](https://github.com/mvark/WebApps/tree/main/ThingsNearMe) uses the GeoNames API to find landmarks / interesting places around you while you're travelling.
19. [Whats In A Name](https://github.com/mvark/WebApps/tree/main/WhatsInAName) is a simple Flask app that uses the Sonar API from Perplexity.ai to fetch interesting, cultural, and historical facts about names.
20. [Where To Watch](https://github.com/mvark/WebApps/tree/main/WhereToWatch) is a simple tool to search for movies using the TMDB API, and check their streaming availability on popular Indian OTT platforms like Netflix, Prime Video, SonyLIV, Hotstar, and Zee5.
